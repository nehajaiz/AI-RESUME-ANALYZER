# 🤖 AI Resume Analyzer

An AI-powered Resume Analysis Platform that helps job seekers optimize their resumes for Applicant Tracking Systems (ATS), identify skill gaps, and improve their chances of getting shortlisted by recruiters.

## 🚀 Overview

AI Resume Analyzer is designed to evaluate resumes against industry standards and job requirements. The platform uses Natural Language Processing (NLP) and Machine Learning techniques to analyze resume content, extract key information, calculate ATS scores, and provide personalized recommendations for improvement.
This project aims to bridge the gap between candidates and recruiters by helping users create ATS-friendly resumes that stand out in today's competitive job market.

---

## ✨ Key Features

### 📄 Resume Parsing

* Extracts information from resumes automatically
* Identifies personal details, education, skills, projects, and experience
* Supports structured resume analysis

### 🎯 ATS Score Analysis

* Calculates resume compatibility score
* Evaluates formatting and keyword relevance
* Highlights missing sections and improvements

### 🧠 Skill Gap Detection

* Compares candidate skills with job requirements
* Identifies missing technical and soft skills
* Suggests areas for improvement

### 🔍 Job Matching

* Matches resumes with relevant job roles
* Provides compatibility insights
* Improves candidate-job alignment

### 📊 Detailed Analytics Dashboard

* Resume score visualization
* Strength and weakness analysis
* Actionable recommendations

### ⚡ Real-Time Feedback

* Instant resume evaluation
* Suggestions for optimization
* Recruiter-focused recommendations

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/nehajaiz/AI-RESUME-ANALYZER.git
cd AI-RESUME-ANALYZER
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

```bash
# Windows
.venv\Scripts\activate

# Linux/Mac
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Backend

```bash
python main.py
```

### Run Frontend

```bash
cd frontend
npm install
npm start
```
